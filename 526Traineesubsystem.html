View Feedback Notes: contains the details required to  into the FICS CORP System.

Navigation Process: By clicking on the  Button on the Static Web page 


Layout:


Controls Description:

Screen Name: View Feedback

Item Number:

Item Name:

Item Description:

1

Session Details (Label)

The session name and number that has been selected is displayed.

2

Search Bar (Search Bar)

This search bar can be used to search for the session name or number to simplify the search.

3

Search (Button)

This will initiate the search and validate the feedback with the searched item.

4

Feedback (Label)

This will display the feedback from the session, written by the trainer or practitioner.

5

Return (Button)

This will return the user to the previous screen, view feedback.

6

Cancel (Button)

This will cancel the operation and will return the trainee to their main dashboard.

7

Search (Menu Item)

This will display a search bar for the trainee to search for a specific thing on the website.

8

Tasks (Menu Item)

This will display all the tasks that have been assigned to the trainee that is still due.

9

Profile (Menu Item)

The trainee can maintain their profile from this menu item.

10

More (Menu Item)

This menu item will display more options for the user.
